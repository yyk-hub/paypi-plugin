How It Works

1

Deploy to Cloudflare Pages

Connect GitHub repository to Cloudflare Pages. Deploy in 2 minutes.

2

Configure Environment

Set your Pi API key and app wallet secret in Cloudflare dashboard.

3

Deposit Credits

Deposit π to get processing credits. 1π = 1 credit. Process 50π per credit deposited.

4

Add to Your Website

One <script> tag and a <div>. Payment button appears automatically.

Integration Example

<!-- Add to your checkout page -->
<script src="https://sdk.minepi.com/pi-sdk.js"></script>
<script src="https://your-paypi.pages.dev/sdk/v1/paypi.js"></script>

<div data-paypi-amount="10" 
     data-paypi-order="ORD-12345"
     data-paypi-description="Premium Plan">
</div>

<!-- That's it! Payment button appears automatically -->
